# Contributing

## Prerequisites

- Windows 10 or later, x64
- .NET 10 SDK
- Visual Studio 2022 Build Tools with the **Desktop development with C++** workload

## Build and test

Run from PowerShell:

```powershell
./scripts/build.ps1
```

Keep changes focused, add tests for managed behavior, and verify both the C++
Release build and the Native AOT publish before opening a pull request.
